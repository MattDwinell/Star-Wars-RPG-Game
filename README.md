# Star-Wars-RPG-Game
1. Welcome to star wars RPG, a javascript game utilizing the jquery library that allows the user to pick a character and fight for whatever side of the force the ystand for!
![start screen](/assets/images/starwars_start.png)
2. to play the game, click on the characters and the attack/restart button.
3. There are a few easter eggs hidden in the code, see if you can find them!
